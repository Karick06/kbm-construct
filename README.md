# KBM Construct Business Management System

## Overview
A business management system starter with focused modules:

- Dashboard and analytics
- CRM and client relationships
- Projects and tasks
- Invoicing and documents
- Team and reports
- Settings and workspace controls

## Tech Stack
- Next.js App Router
- TypeScript
- Tailwind CSS

## Getting Started
```bash
npm install
npm run dev
```

Open the app at `http://localhost:3000`.

## Scripts
```bash
npm run dev
npm run build
npm run start
npm run lint
```

## Notes
This scaffold ships with sample data and UI-only flows. Replace the sample data with real services and a database when ready.
