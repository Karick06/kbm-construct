# BoQ System Quick Start Guide

## Getting Started

### 1. Creating Your First BoQ

**Home → Estimating → BoQ Overview → Create BoQ**

### 2. Choose Your Standard

#### For Building Projects (Houses, Offices, Commercial)
→ Use **SMM7** templates

#### For Civil Engineering (Roads, Tunnels, Bridges)
→ Use **CESMM** templates

#### For Highways and Roads (Motorways, Rural Roads, Parking)
→ Use **SHW** templates

---

## Using Templates (Recommended)

### Step-by-Step

1. Go to **Estimating → BoQ Templates**
2. Select your measurement standard (SMM7/CESMM/SHW)
3. Browse available templates for your project type
4. Click **"Use Template"**
5. Enter:
   - Project name
   - Your name (prepared by)
6. Review and add/remove items as needed
7. Input rates and quantities
8. Add contingency percentage (default 10%)
9. Save or export

## Common Use Cases

### Use Case 1: Residential Building Estimate (SMM7)

**Project:** New house extension

**Standard:** SMM7 (Building works)

**Template:** "Building Extension"

**Typical Sections:**
- Preliminaries (£20,000)
- Ground works (£15,000)
- Concrete foundations (£12,000)
- Steelwork/Timber framing (£8,000)
- Roofing (£18,000)
- Cladding/brickwork (£35,000)
- Internal finishes (£22,000)
- Services (£25,000)

**Total:** ~£155,000 + 10% contingency = £170,500

## Item Entry Examples

### SMM7 Example: Brickwork

```
Item Number: E.1.1
Description: External brickwork, 102.5mm single skin, London stock bricks, 
             mortar pointing, per m²
Unit: m²
Quantity: 500 (measured from drawings)
Rate: £45 (per m², includes labour, materials, plant)
Amount: £22,500
```

## Unit Selection Guide

| What You're Measuring | Standard Unit | Example |
|-----------------------|---------------|---------|
| Wall surface area | m² | "external walls per m²" |
| Excavation/fill | m³ | "soil excavation per m³" |
| Items/fittings | no | "doors per no" / "windows per no" |
| Roofing | m² | "tile roof covering per m²" |
| Lump sum items | sum | "preliminaries sum" |
| Structural elements | t | "steel reinforcement per t" |
| Linear runs | m | "drainage pipes per m" |
| Complete rooms | no | "consulting rooms per no" |

## Contingency Guidelines

### Uncertain Factors:

**5-10% Contingency:**
- Well-defined scope
- Detailed drawings available
- Similar past projects
- Low risk profile

**10-15% Contingency:**
- Moderate detail available
- Some unknowns (ground conditions, access)
- Typical mid-stage estimates
- Medium risk

**15-20% Contingency:**
- Early stage estimate
- Some scope uncertainty
- Historical data limited
- Higher risk factors

**20%+ Contingency:**
- Very early pre-design
- Major unknowns (unexplored site, new technology)
- Experimental work
- High complexity

## Quality Checklist

Before submitting your BoQ:

- [ ] All items have clear descriptions
- [ ] Quantities verified against drawings/specifications
- [ ] All rates researched and realistic
- [ ] Units correct and consistent
- [ ] Sections complete (no major items missing)
- [ ] Contingency percentage reasonable
- [ ] Total amount sensible for project type
- [ ] Project details filled in (name, date, preparer)
- [ ] No duplicate items
- [ ] No contradictory items

## Common Mistakes to Avoid

❌ **Incomplete descriptions**
✓ Be specific: "brickwork per m²" is vague; "102.5mm external brickwork with pointing per m²" is clear

❌ **Unrealistic rates**
✓ Check current market: too low means project failed; too high means client rejects

❌ **Missing items**
✓ Review specification thoroughly; consult with team about potential items

❌ **Inconsistent units**
✓ Don't mix "no" and "nr" - be consistent throughout

❌ **No contingency**
✓ Always include 5-20% depending on risk

❌ **Quantity errors**
✓ Double-check measurements; get peer verification for large amounts
